# Flashcards Química

Una aplicación interactiva para dominar conceptos fundamentales de química, como la ley de los gases, conversiones y constantes.

## Características
- 🧪 **Tarjetas Interactivas**: Repasa leyes y fórmulas con un sistema de tarjetas que se pueden girar.
- 🔀 **Modo Examen**: Pon a prueba tus conocimientos con un modo de examen aleatorio.
- 🌓 **Modo Oscuro**: Interfaz adaptable para estudiar en cualquier momento.
- 📊 **Seguimiento de Progreso**: Visualiza cuántas tarjetas has revisado.

## Instalación
Puedes descargar el APK de la última versión en la sección de [Releases](https://github.com/maxiusofmaximus/Flashcards-Quimica/releases).

## Desarrollo
Este proyecto está desarrollado con **Jetpack Compose** y sigue las mejores prácticas de UI/UX de Android.

### Requisitos
- Android 7.0 (API 24) o superior.
- Gradle 8.0+.
