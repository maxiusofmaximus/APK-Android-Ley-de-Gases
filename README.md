## Características

- 🧪 **25 Tarjetas Interactivas** — Cubren conversiones, constantes, leyes y fórmulas
- 🔀 **Modo Examen** — Pon a prueba tus conocimientos con preguntas aleatorias
- 📊 **Seguimiento de Progreso** — Visualiza cuántas tarjetas has revisado
- 🏷️ **Filtrado por Categorías** — Conversiones, Constantes, Leyes, Fórmulas, Solubilidad
- 🌓 **Tema Claro/Oscuro** — Interfaz adaptable para estudiar en cualquier momento
- ✨ **Animaciones Premium** — Experiencia fluida e interactiva

## Contenido de las Tarjetas

### Conversiones
| Unidad | Conversión |
|--------|------------|
| 1 mmHg | 1 torr |
| 1 atm | 760 mmHg |
| 1 atm | 101,325 Pa |
| 1 Pa | 9.86 × 10⁻⁶ atm |

### Constantes
| Constante | Valor |
|-----------|-------|
| Temperatura Normal | 273.15 K |
| Presión Normal | 1 atm |
| Volumen Normal | 22.4 L |
| Número de Avogadro | 6.022 × 10²³ |
| Constante R | 0.082 L·atm/mol·K |

### Leyes
- **Ley de Boyle**: V₁P₁ = V₂P₂ (T y n constantes)
- **Ley de Charles**: V₁/T₁ = V₂/T₂ (P y n constantes)
- **Ley de Gay-Lussac**: P₁/T₁ = P₂/T₂ (V y n constantes)
- **Ley de Dalton**: PT = P1 + P2 + P3...
- **Gas Ideal**: PV = nRT

## Instalación

### Web
Simplemente abre `index.html` en tu navegador o visita la [demo en vivo](https://maxiusofmaximus.github.io/Flashcards-Quimica/).

### Android
Descarga el APK desde la sección de [Releases](https://github.com/maxiusofmaximus/Flashcards-Quimica/releases) o compila el proyecto tú mismo:

```bash
cd android
./gradlew assembleRelease
```

## Desarrollo
