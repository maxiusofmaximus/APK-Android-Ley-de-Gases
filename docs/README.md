# Documentación

## Proyecto Flashcards Química

Este repositorio contiene una aplicación de tarjetas educativas interactivas para aprender química, específicamente la Ley de los Gases.

## Estructura del Proyecto

```
├── index.html          # Página web principal (QuímicaCards)
├── css/
│   └── styles.css      # Estilos de la página web
├── js/
│   └── app.js          # Lógica de la aplicación web
├── android/            # Aplicación Android (Jetpack Compose)
│   └── README.md       # Documentación de la app Android
└── docs/               # Documentación del proyecto
```

## Página Web

La página web está ubicada en la raíz del proyecto y es servida a través de GitHub Pages.

### Características
- Tarjetas interactivas con animaciones flip
- Modo examen con preguntas aleatorias
- Sistema de filtrado por categorías
- Tema claro/oscuro
- Seguimiento de progreso

### Categorías de Tarjetas
- **Conversiones**: mmHg, atm, Pa, torr
- **Constantes**: Temperatura, presión, volumen normal, Avogadro
- **Leyes**: Boyle, Charles, Gay-Lussac, Dalton
- **Fórmulas**: Gas ideal, densidad, peso molecular
- **Solubilidad**: Reglas de solubilidad de compuestos

## Aplicación Android

La aplicación Android está desarrollada con Jetpack Compose. Consulta `android/README.md` para más detalles.
